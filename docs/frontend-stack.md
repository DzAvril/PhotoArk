# WebUI 技术栈（2026 初始化版本）

## 栈选型
- React + TypeScript + Vite
- Tailwind CSS（设计令牌 + 响应式）
- Framer Motion（关键动效）
- vite-plugin-pwa（PWA 安装、离线缓存、SW 自动更新）

## 设计目标
- PC / 平板 / 手机统一信息架构
- 视觉方向：自然色系 + 磨砂卡片 + 渐变背景层次
- 可读性优先：高对比标题、紧凑信息卡片、移动端单列布局

## PWA 能力
- 可安装到桌面和手机主屏
- Service Worker 自动更新
- 静态资源离线缓存
- 自定义 Manifest 和图标
